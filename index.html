NaturalCC Code Agent

把静态项目理解能力和 Aider 的文件修改能力组合在一起的本地代码编辑代理。

针对 C/C++ 等工程化项目，先解析目标项目收集上下文，生成语义增强 Prompt，再交由 Aider 精准修改目标文件。

Launch Web UI View Documentation

核心特性

为 C/C++ 工程化项目量身打造的智能代码编辑能力

精准的静态解析

依赖 libclang 深入解析 C/C++ 项目，收集函数、变量、类型和符号关系。

语义增强 Prompt

自动构造面向项目上下文的语义 Prompt，并在执行 Aider 前支持预览最终形态。

灵活的运行模式

提供基于 FastAPI + React/Vite 的可视化 Web UI，以及轻量级的命令行模式。

典型任务

NaturalCC Code Agent 可以帮你高效完成以下开发任务

补全函数体

根据函数签名和项目上下文，自动生成完整的函数实现。

补全函数签名

基于调用点和类型推断，补全缺失的函数参数与返回类型声明。

补全变量或类型

自动推断并补全代码中缺失的变量声明和类型定义。

按项目现有风格做小范围代码修改

分析项目既有编码风格，在保持风格一致的前提下进行精准的局部代码调整。

架构设计

模块化架构，易于扩展和自定义

规划层 (Planner)

接收自然语言指令，分析项目上下文，制定分步执行计划。支持复杂任务拆解和依赖分析。

执行层 (Executor)

按照计划执行具体编辑操作，包括文件读写、代码插入/删除/替换，支持原子化回滚。

验证层 (Validator)

自动运行语法检查、单元测试，确保修改后的代码正确可用，防止引入回归问题。

接口层 (Interface)

提供 CLI 命令行工具和 FastAPI + React/Vite Web UI，灵活集成到各种工作流。
